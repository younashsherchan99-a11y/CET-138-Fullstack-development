Bootstrap Basics

Bootstrap is a free front-end framework for faster and easier web development.
It includes HTML and CSS-based design templates for typography, forms, buttons, tables, navigation, modals, image carousels, and many others, as well as optional JavaScript plugins.
Bootstrap allows developers to easily create responsive designs.

Bootstrap Versions:
Bootstrap 5 (released 2021) is the newest version, with new components, faster stylesheet, and more responsiveness.
Bootstrap 5 supports the latest stable releases of all major browsers and platforms (Internet Explorer 11 and older is not supported).
The main difference from Bootstrap 3 & 4 is that Bootstrap 5 uses vanilla JavaScript instead of jQuery.

Summary

Bootstrap is a front-end framework that provides ready-to-use components like buttons, forms, tables, navigation, and more.
Bootstrap makes web development faster and helps create responsive websites.
Bootstrap 5 uses vanilla JavaScript instead of jQuery, providing lighter and modern code.
Use Bootstrap by linking its CSS & JS in your HTML file, and then use classes like btn, alert, container etc.
