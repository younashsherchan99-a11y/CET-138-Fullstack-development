CSS Basics

CSS is the language we use to style a web page.
CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are displayed on screen, paper, or other media.
CSS saves a lot of work because it can control the layout of multiple web pages at once.
External stylesheets are stored in separate CSS files.

Example CSS

1. Inline CSS

<p style="color:red; font-weight:bold;">
This is an inline CSS paragraph.
</p>

This is an inline CSS paragraph.

2. Internal CSS

/* Internal CSS example */
p.internal {
  color: blue;
  font-size: 18px;
}

This is an internal CSS paragraph.

3. External CSS

/* External CSS example */
.external {
  color: green;
  font-style: italic;
}

This paragraph is styled using external CSS.

Summary

CSS can be applied in three ways:
1. Inline CSS – directly in HTML element using style.
2. Internal CSS – inside <style> tag in head.
3. External CSS – stored in separate CSS file linked using <link>.
